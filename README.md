**Project Overview :**
This project automates the workflow of a data analyst. It generates a year's worth of sales data for three products, processes that data to find key business insights, and creates visual reports.

**Key Features :**
Automated Data Generation: Uses NumPy to create realistic monthly sales figures.

**Performance Tracking :**
Calculates total annual sales and identifies the highest and lowest performing months.

**Product Comparison :**
Determines which product contributed most to the total revenue.

**Visualization :**
Generates and saves three types of charts: Monthly Trends, Product Comparison, and Total Annual Volume.

**Tech Stack :**
Language: Python

Libraries: Pandas, NumPy, Matplotlib, Seaborn

**How to Use :**
Clone the repository to your local machine.
Install dependencies:
pip install pandas numpy matplotlib seaborn

**Run the script :**
python sales_analysis.py
View Results: The script will print statistics to the console and save visual charts in a folder named /charts
